入れ子(ネスト)構造

HTMLのように入れ子構造で書くことができます。要素の親子関係がわかりやすくなるうえに、同じセレクタを何度も書く必要がありません。

CSS


.top-ttl {
  margin: 0 auto;
}
.top-ttl h1 {
  font-weight: 900;
  letter-spacing: 4px;
}
.top-ttl h1 span {
  color: #C6538C;
}

Sass


.top-ttl {
  margin: 0 auto;
  h1 {
    font-weight: 900;
    letter-spacing: 4px;
    span {
      color: #C6538C;
    }
  }
}

変数が使える

他の多くのプログラミング言語と同様に変数を使うことができます。これによって、スムーズなコーディングとメンテナンスが実現できます。

CSS


h1 {
  color: #C6538C;
  font-size: 2em;
}
p {
  color: #C6538C;
  font-weight: 900;
}

Sass


$color: #C6538C;
h1 {
  color: $color;
  font-size: 2em;
}
p {
  color: $color;
  font-weight: 900;
}

@mixinと@include

@mixinでスタイルをテンプレートのようにあらかじめ定義しておくことで、@includeを使ってあとからいつでも呼び出すことができます。

CSS



.boxA {
  color: #fff;
  font-size: 2em;
}
.boxB {
  margin: 0 auto;
  color: #fff;
  font-size: 1.5em;
}

Sass


@mixin box($fontSize: 2em) {
  color: #fff;
  font-size: $fontSize;
}
.boxA {
  @include box;
}
.boxB {
  margin: 0 auto;
  @include box(1.5em);
}
